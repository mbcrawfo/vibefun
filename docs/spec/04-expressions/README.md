# Expressions

Vibefun expressions form the core computational units of the language. Every expression has a type and produces a value.

## Overview

Vibefun uses expression-oriented syntax where nearly everything is an expression that returns a value, including:
- Control flow constructs (`if`, `match`)
- Blocks (`{ ... }`)
- Pattern matching
- Function definitions

This design enables functional composition and encourages declarative programming style.
