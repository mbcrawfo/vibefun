# Basic Expressions

### Literal Expressions

```vibefun
42          // Int
3.14        // Float
"hello"     // String
true        // Bool
()          // Unit
```

### Variable References

```vibefun
let x = 42
let y = x  // Variable reference
```

### Function Calls

```vibefun
add(1, 2)              // Function call
map([1, 2, 3], double) // Higher-order function
```

### Operators

```vibefun
1 + 2                  // Arithmetic
x == y                 // Comparison
true && false          // Logical
"hello" & " world"   // String concatenation
```
