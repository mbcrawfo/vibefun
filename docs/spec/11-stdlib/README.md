# Standard Library

The Vibefun standard library provides essential data structures and utilities for functional programming.

5. **[Numeric](./numeric.md)** - Int and Float operations

7. **[Collections](./collections.md)** - Map and Set data structures
8. **[Math](./math.md)** - Mathematical functions
9. **[JSON](./json.md)** - JSON parsing and serialization

## Overview

All standard library functions are:
- **Pure**: No side effects (except Array mutations)
- **Type-safe**: Fully typed with inference support
- **Curried**: All multi-argument functions are curried by default
- **Composable**: Designed for use with pipes and function composition
